# laggy

### Commands

```
laggy lag ipv4
```

```
laggy lag ipv6
```

```
laggy dev ipv4
```

```
laggy dev ipv6
```

### Control Tests

```
*.lag.4n6ir.com
```

vs.

```
*.dev.4n6ir.com
```

### Output Results

```
site: https://ipv4.use1.lag.4n6ir.com, Status: 200, Latency: 119.185753ms
```

Results if IPv6 is not supported.

```
site: https://ipv6.afs1.lag.4n6ir.com, Status: 0, Latency: 0ns
```
